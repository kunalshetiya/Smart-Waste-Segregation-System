# ♻️ Smart Waste Segregation System

An AI-powered system that automatically classifies and segregates waste into **organic, recyclable, and hazardous categories** using sensors and computer vision.

## 🚀 Features

* AI-based waste classification
* Smart conveyor sorting system
* IoT + sensor integration
* Web dashboard & mobile app
* User reward & incentive system

## 🛠️ Tech Stack

* Python, TensorFlow, OpenCV
* Raspberry Pi, Sensors, Motors
* Cloud + Web Dashboard

## 🌍 Impact

* Reduces landfill waste
* Improves recycling efficiency
* Promotes clean and sustainable environment
